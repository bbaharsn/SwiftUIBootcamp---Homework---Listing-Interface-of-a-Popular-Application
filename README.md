<h2>Step 3</h2>
<p>This is the third version of the interface. At this stage, I built the personalized product listing section of the home page. Each product card displays the product image, name, star rating, review count, and price. Discounted products show the original price with a strikethrough and highlight the new price in a distinct color, along with a discount badge (e.g. -%20). Products with free shipping display a gray banner at the bottom of the image. A favorite button with a heart icon is placed on the top right corner of each product image. A "Tümünü Gör" (See All) button is included in the section header to navigate to the full product list. The cards are displayed in a horizontally scrollable row using a ForEach loop connected to the product model.</p>

<h2>Step 4</h2>
<p>This is the fourth version of the interface. At this stage, I added the personalized category section with two horizontally scrollable rows of category cards, each displaying an image and label. I also implemented the bottom navigation bar with five tabs: Anasayfa, Trendyol Go, Favorilerim, Sepetim, and Hesabım. Additionally, the gradient background behavior was fixed so that it fades away when the user scrolls down, leaving a clean white header with only the search bar visible.</p>
